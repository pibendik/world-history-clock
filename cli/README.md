# Historieklokka — terminal

Viser hva klokken ser ut som som et historisk årstall og en hendelse fra det året.

**15:23** → år **1523** → «Luther utgir *Om verdslig øvrighet*»

Data hentes fra [historieklokka.no](https://historieklokka.no).

## Installasjon

```bash
pip install historieklokka
```

## Bruk

```bash
historieklokka
historieklokka --api http://localhost:8421/api/v1   # mot lokal instans
```

Avslutt med `Ctrl+C`.
